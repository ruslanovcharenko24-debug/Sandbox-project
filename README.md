# Sandbox Projects

Репозиторий для учебных и личных проектов по разработке.

## Структура

```
sandbox-project/
├── README.md
└── .gitignore
```

## Правила

- Учебные материалы (лекции, презентации) хранятся **вне этой папки**:
  - `SANDBOX/MTU/` — материалы MTU
  - `SANDBOX/IGSU/` — материалы ИГСУ
- В этот репозиторий попадает **только код и проектные файлы**
- Бинарники (PDF, DOCX, PPTX, видео, архивы) исключены через `.gitignore`
